# Java Password Generator 🔐

## Overview

This project is a simple **Password Generator written in Java**.
The program generates a random password based on user preferences such as including **uppercase letters, lowercase letters, and special characters**.

The goal of this project was to practice **Java fundamentals and logical problem-solving**.

---

## Features

* User chooses the **length of the password**
* Option to include **special characters**
* Option to include **uppercase letters**
* Option to include **lowercase letters**
* Random password generation using Java's `Random` class
* Handles invalid input when no character type is selected

---

## Technologies Used

* Java
* Scanner class for user input
* Random class for generating random characters

---

## How It Works

1. The user enters the desired password length.
2. The program asks whether to include:

   * Special characters
   * Uppercase letters
   * Lowercase letters
3. Based on the user's choices, the program selects characters from predefined character sets.
4. Random characters are chosen to generate the final password.
